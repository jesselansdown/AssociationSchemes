  [33X[1;0Y[29X[2XSaveHomogeneousCoherentConfigurationWithCertainAttributes[102X( [3Xfile[103X, [3XA[103X, [3XL[103X ) [32X operation[133X

  [33X[0;0YSaves  homogeneous  coherent  configuration  A to file F with the attributes

  attribute  known  for  A.  Note  that  Print or PrintTo will only return the

  [1X4.1 [33X[0;0YExample 1 -- Constructing groups[133X[101X
  
  [33X[0;0YIn this example, we show how we can use coherent configurations to construct
  an  entriely  different almost simple permutation group from another one. We
  first show how [23XPSU(4,3)[123X can be made out of its subgroup [23XPSL(3,4)[123X.[133X

  [1X4.5 [33X[0;0YExample 5 -- Constructing HS (advanced example)[133X[101X
  
  [33X[0;0YWe  redo  an  example  that  appears  in  Section  3.6  of  Peter  Cameron's

  [33X[0;0YFirst we construct the Hoffman-Singleton graph from the alternating group of

  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XA7 := AlternatingGroup(7);;[127X[104X
    [4X[25Xgap>[125X [27XPi := [ [ 1, 2, 4 ], [ 1, 3, 7 ], [ 1, 5, 6 ], [127X[104X
    [4X[25X>[125X [27X [ 2, 3, 5 ], [ 2, 6, 7 ], [ 3, 4, 6 ], [ 4, 5, 7 ] ];;[127X[104X
    [4X[25Xgap>[125X [27XOnSetsRecursive := function(x,g) [127X[104X
    [4X[25X>[125X [27X	if not IsSet(x) then[127X[104X
    [4X[25X>[125X [27X		return x^g; [127X[104X
    [4X[25Xgap>[125X [27X	else [127X[104X

    [4X[25Xgap>[125X [27X	fi;[127X[104X
    [4X[25Xgap>[125X [27Xend;;[127X[104X

    [4X[25Xgap>[125X [27XallFanos := Orbit(A7, Pi, OnSetsSets);;[127X[104X

    [4X[25Xgap>[125X [27XList(orbitals, Size);[127X[104X
    [4X[28X[ 210, 315, 70, 420, 105, 105 ][128X[104X
